# E-commerce (React + Vite + TS + Supabase)

E-commerce de celulares construido con **React**, **Vite** y **TypeScript**, usando **Supabase** como backend (Auth + Database). Incluye catálogo público, detalle por `slug`, carrito, checkout y panel de pedidos por usuario.

Proyecto personal en progreso (≈60%). Enfocado en buenas prácticas de frontend moderno, tipado fuerte y arquitectura escalable.

## Features

### Catálogo (público)

- Listado de celulares
- Detalle de producto por `slug`
- Variantes (color / almacenamiento / stock)
- Búsqueda / filtrado (según implementación actual)

### Usuario

- Registro e inicio de sesión con Supabase Auth
- Sección de cuenta
- Historial de pedidos
- Detalle de pedido

### Checkout

- Flujo de checkout
- Página de confirmación (`thank-you`)

---

## Tech Stack

- **Frontend:** React, Vite, TypeScript
- **Routing:** React Router
- **State:** Zustand
- **Server state:** TanStack Query
- **Forms:** React Hook Form + Zod
- **Backend:** Supabase (Postgres + Auth)
- **UI:** Tailwind CSS, React Icons, React Hot Toast
- **Editor:** TipTap
